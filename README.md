## About 
Allow control of Sonos speakers from Mycroft.
Assumes all speakers are in one group.

## Examples 
* "Sonos play"
* "Sonos pause"
* "Sonos skip"
* "Sonos volume up"
* "Sonos volume loud"
* "Sonos volume middle"
* "Sonos volume down"
* "Sonos volume soft"
* "Sonos what song is playing"
